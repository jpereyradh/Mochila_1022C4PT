¿Quién inventó el sistema de control de versión Git y por qué?
Es un proyecto de codigo abierto desarrollado originalmente por Linus Torvalds. Es un sistema de control de versiones que permite comparar el código de un archivo para ver las diferencias entre las versiones, restaurar versiones antiguas si algo sale mal, y fusionar los cambios de distintas versiones
