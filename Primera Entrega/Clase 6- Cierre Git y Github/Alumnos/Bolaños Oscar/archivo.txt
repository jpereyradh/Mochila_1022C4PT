a. ¿Quién inventó el sistema de control de versión Git y por qué?

Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005.

Se inventó porque este software ayuda a los equipos a resolver los problemas a la hora de realizar un un seguimiento de todos los cambios individuales de cada colaborador y que al contribuir al repositorio se evita que el trabajo concurrente entre en conflicto.
