¿Quién inventó el sistema de control de versión Git y por qué?
Git ideado como un proyecto de código y control de versiones fue creado por Linus Torvalds
Por que el gestor de versiones que poseia antes linux paso a ser pago y Linus Torvalds decidio crear su propio software de control de versiones.
